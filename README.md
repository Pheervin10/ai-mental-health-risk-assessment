# AI-Based Mental Health Risk Assessment System

This project aims to develop an AI-based system to assess mental health risk levels using user input data and machine learning techniques.

## Project Scope
- Collect mental health-related input data
- Analyze risk patterns using AI models
- Provide risk assessment results via a simple interface

## Tools Used
- Python
- Flask
- Scikit-learn
- GitHub

## Project Management
This project is managed using Microsoft Project following standard AI project management process groups.
